Up Mini jeu | Site web Découvrir le jeu

Dans le cadre de ma formation en MMI, j'ai été amené à réaliser un mini jeu à l'aide de la librairie javascript D3.js. Nous devions créer un jeu du type "Shoot'em Up" dont le but est de gagner le plus de points possible avec un nombre limité de vies.


Grâce à ce projet, j'ai pu découvrir et acquérir des compétences en programmation par objet (POO). J'ai réalisé ce jeu en binôme avec Merwan Jaudally pour lequel nous nous sommes répartis les tâches équitablement.
